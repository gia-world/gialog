# GiaLog

Next.js로 프론트 파트와 서버 구현까지 직접 만들어 본 블로그입니다.

1. 파일 시스템의 마크다운 파일을 이용하여 조회, 추가, 수정, 삭제 등의 서비스로직 구현
2. Next.js의 route handler로 API 구현
3. 상태관리 : 리덕스

## 작업기간

2023.11.03 ~

## 기능

- ==Home==
  - 고정 게시글 3개
  - 전체 게시글 캐러설
- ==About==
  - 소개 페이지
- ==Posts==
  - 게시글 목록
    - 고정 게시글 추가 및 삭제 기능
  - 게시글 작성
  - 상세페이지
    - 수정 및 삭제 기능
- ==Contact==
  - 연락처 정보
    - 이메일 전송 기능

## 사용 스택 및 라이브러리

- Next.js 13
- Tailwind
- React-icon
- React-slick
- React-markdown, gray-matter
- Axios
- Redux & RTK
- React-hook-form
- nodemailer
