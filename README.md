# GiaLog

직접 만들어 본 블로그입니다.

## 작업기간

2023.11.03 ~

## 기능

- 마크다운 파일을 이용하여 블로그
- 상태관리 : 리덕스
- ==Home==
  - 지정한 고정 게시글 3개 보여주기
  - 전체 게시글 캐러설
- ==About==
  - 소개 페이지
- ==Posts==
  - 전체 게시글 목록
  - 게시글 작성
  - 상세페이지에서 삭제 가능
- ==Contact==
  - 이메일 전송 가능

## 사용 스택 및 라이브러리

- Next.js 13
- Tailwind
- React-icon
- React-slick
- React-markdown
- Axios
- RTK
